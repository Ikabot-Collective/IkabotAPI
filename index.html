What is this?
The above code is a browser fingerprinting token (referred to as a "blackbox" token) that was just generated by your browser. The purpose of the token is to ensure that players are using an actual browser when playing. The code is generated using an obfuscated JavaScript file (hosted at https://gameforge.com/tra/game1.js).

This JavaScript code consolidates information such as:
  • The current date and time
  • Browser's version and user agent
  • Host OS type and version
  • Fonts available for rendering
  • Display device and graphics processor
  • Browser window size
  • And much more...

All this is combined into a single blackbox token that proves it came from a legitimate browser.
